Why this exists

Designing UI colors is not just about choosing "nice colors."

It requires:

  • consistency across roles
  • readability and contrast
  • balance between usability and expression

This app translates those design decisions into a system.

What you can do

  • Generate 5 distinct palette directions
    Baseline / Clarity / Expression / Serene / Impact
  • Work with 6 structured color roles
    Primary Accent (Seed) / Secondary Accent / Background / Surface / Text / Border
  • Adjust tone using
    Warmth / Saturation / Depth
  • Control priority
    Style / Usability / Accessibility
  • Shuffle the Secondary Accent while keeping the Primary fixed to explore palette variations.

Built for real UI

Each palette is designed with real usage contexts in mind, taking into account the specific requirements of each scenario.

  • Web Page
    Designed for readability across long content. Optimized for text contrast, link visibility, and balanced emphasis.
  • Mobile App
    Designed for compact screens and touch interaction. Optimized for clear hierarchy, component separation, and quick recognition.
  • Presentation
    Designed for projected and slide-based viewing. Optimized for strong contrast, legibility at distance, and visual clarity.
  • Poster
    Designed for visual impact and quick attention. Optimized for bold contrast and strong focal points, while also considering print-friendly color balance to avoid excessive saturation.
  • Magazine
    Designed for editorial layouts and extended reading. Optimized for tonal balance and reduced visual fatigue, with adjustments to maintain stable and natural color reproduction in print.

Accessibility approach

  • Baseline guarantees WCAG AA contrast
  • Other patterns allow more flexibility for visual expression

You can explore both safe and expressive options side by side.

Color vision simulation

The app includes an approximate simulation for color vision differences:

  • P-type (Protanopia)
  • D-type (Deuteranopia)

This allows you to preview how your palettes may appear to users with different color perception, helping you identify potential issues in distinction and contrast early in the design process.
